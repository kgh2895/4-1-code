## Important Rules

### Rules (1/3)

- **Late submission** of the topic title (after the announced date) will result in a **loss of marks**.
- **Final submissions after the deadline will NOT be accepted.**
- Each project can be done by **1–2 students**.
- The selected project idea must be **unique** and based on your own interest.
  - If two teams choose the same idea, **priority will be given to the first submission**.
- The project **code and concept must work correctly and successfully**.

### Rules (2/3)

- The project idea must **NOT be too simple or trivial**.
- You must **demonstrate a full understanding** of the project through detailed explanations.
- Prepare a **PDF report of at least 7 pages**.
- The report must be written in **English** and include:
  - All relevant code
  - Step-by-step screenshots to clearly explain the implementation
- A **demo video** is required and should be **short (10–20 seconds)**.

### Rules (3/3) — Summary PDF Should Include

- A clear description of the **main idea** of the project
- A **detailed explanation** (with figures/diagrams) covering:
  - Project goals
  - Design and methodology
  - Implementation/code
  - Results and outcomes
- A description of the **tools and software** used
- A **discussion** of the key findings and main benefits of the project

## Suggested Ideas for Assembly Projects *(Not limited to)*

### Beginner-Level Projects *(individual — must add an extension)*

| Project | Description |
|---|---|
| **Simple Calculator** | Performs basic arithmetic operations: addition, subtraction, multiplication, and division. |
| **Temperature Converter** | Converts temperature between Celsius and Fahrenheit based on user input. |
| **Odd or Even Number Checker** | Checks if a number is odd or even using modulus operations and conditional branching. |
| **LED Blinking** | Basic LED toggle with delays. |
| **String Length Calculator** | Count characters in a string without using C functions. |
| **Factorial Calculator** | Calculate factorial using loops or recursion. |
| **Show Text on an LCD** | Write text on an LCD screen using Assembly (use simulators like Proteus). |

### Intermediate-Level Projects *(individual or team)*

| Project | Description |
|---|---|
| **Digital Clock** | Display a running clock using timers. |
| **Game (Snake or Ping Pong)** | Build basic games using graphics memory. |
| **Timer with LCD** | Countdown timer — user inputs time in seconds, decrements to zero, then displays a message. |
| **Basic Graphics on LCD** | Draw simple 2D shapes (rectangles, circles) on an LCD screen. |
| **Traffic Light Control** | Program to control the timing of traffic lights. |
| **Heater Alarm** | Monitor boiler temperature every 3 minutes; show alarms on green/red LEDs. |
| **Marathon Results** | Store player numbers and completion times in memory; rearrange in ascending order to find the winner. |
